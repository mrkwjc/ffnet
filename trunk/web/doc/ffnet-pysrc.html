Source Code for Package ffnet

 1  ''' 
 2  ffnet is a fast and easy-to-use feed-forward neural  
 3  network training solution for python. 
 4    
 5  Unique features present in ffnet 
 6  -------------------------------- 
 7  1. Any network connectivity without cycles is allowed. 
 8  2. Training can be performed with use of several optimization  
 9     schemes including: standard backpropagation with momentum, rprop, 
10     conjugate gradient, bfgs, tnc, genetic alorithm based optimization. 
11  3. There is access to exact partial derivatives of network outputs  
12     vs. its inputs. 
13  4. Automatic normalization of data. 
14    
15  Basic assumptions and limitations 
16  --------------------------------- 
17  1. Network has feed-forward architecture. 
18  2. Input units have identity activation function,  
19     all other units have sigmoid activation function. 
20  3. Provided data are automatically normalized, both input and output,  
21     with a linear mapping to the range (0.15, 0.85). 
22     Each input and output is treated separately (i.e. linear map is  
23     unique for each input and output). 
24  4. Function minimized during training is a sum of squared errors  
25     of each output for each training pattern. 
26    
27  Performance 
28  ----------- 
29  Excellent computational performance is achieved implementing core 
30  functions in fortran 77 and wrapping them with f2py. ffnet outstands 
31  in performance pure python training packages and is competitive to 
32  'compiled language' software. Moreover, a trained network can be  
33  exported to fortran sources, compiled and called from many  
34  programming languages. 
35    
36  Usage 
37  ----- 
38  Basic usage of the package is outlined below.  
39  See description of ffnet module and its functions  
40  (and especially ffnet class) for detailed explanations. 
41    
42  >>> from ffnet import ffnet, mlgraph, savenet, loadnet, exportnet 
43  >>> conec = mlgraph( (2,2,1) ) 
44  >>> net = ffnet(conec) 
45  >>> input = [ [0.,0.], [0.,1.], [1.,0.], [1.,1.] ] 
46  >>> target  = [ [1.], [0.], [0.], [1.] ] 
47  >>> net.train_tnc(input, target, maxfun = 1000) 
48  >>> net.test(input, target, iprint = 2) 
49  >>> savenet(net, "xor.net") 
50  >>> exportnet(net, "xor.f") 
51  >>> net = loadnet("xor.net") 
52  >>> answer = net( [ 0., 0. ] ) 
53  >>> partial_derivatives = net.derivative( [ 0., 0. ] ) 
54    
55  Usage examples with full description can be found in  
56  examples directory of the source distribution. 
57  '''
