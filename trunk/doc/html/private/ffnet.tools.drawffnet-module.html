Module ffnet.tools.drawffnet

Network architecture visualizations
Function Summary
  drawffnet(net, biases)
Takes a ffnet class instance and draws the network.

Function Details

drawffnet(net, biases=False)

Takes a ffnet class instance and draws the network. Networkx layouts and maplotlib buttons are used to control layout.

Note: This is draft solution. Function seems not to work with older versions of matplotlib (below 0.87.7, e.g. in enthought python we have 0.87.3 and there are problems). There might occur also problems with graphviz layouts on Windows.
